Pure CTF v2.00 by Methabol - Pure CTF - CTF.SE

Since Zoid's first version of 3wave CTF there have been many different CTF mods out there. Some good and some not so good. Most had one thing in common though: You had to play with all new features, you couldn't just select the ones you want. I wanted a mod where you could select exactly what new features you wanted, and which ones you rather not use at all. I also wanted a solid competition mod that you could use to set up clanfights or practice games quickly and easily. Finally I wanted a mod that fixed all those small but annoying bugs that was left since the developement of 3wave CTF was discontinued. The result is Pure CTF.

With Pure CTF you got the choice. You can select any of the dozens of options, modes and clientpaks you want. You can use it as a competition server, arena server, clanarena server, even a DM server, and it supports the most popular CTF mod besides 3wave, Thunderwalker CTF. And you can combine any of the features making the possibilities endless.

Or you can play just plain, Pure CTF.

Player Settings(v1.51)

Add these settings to your config or type them in the console. New for v1.51 is that all settings is set with one setinfo variable, "pure", to prevent setinfo overflow. Add the values you want from the table and then add a line like this to your config:

setinfo pure 8258 This sets the statusbar to pos. 2 with disabled rune status. and enables autoshot 2+64+8192=8258
   

An easy way to do small changes is to go to a Pure server and use the related commands or the menu to set the settings you want. After each change Pure will tell you the new value of the pure variable for your config. 

NOTE: If you use some of the old variables (ad, as, etc), Pure will acknowledge them when you enter a server and tell you the correct new setting for "pure". After you have added it, remove the old ones. (Avoids trouble.)

Player Settings Command Values
Statusbar position Sets the position of the CTF statusbar. impulse 70-81 0 Statusbar disabled.
1-11 Set position.
Enable auto screenshots Takes a screenshot automatically at the end of a match. autoshot 16 Autoshot enabled.
Enable auto demo Automatically starts and stops demo in matches so you don't get a lot of useless stuff from standby or map changes. autodemo 0 Recording off.
32 Recording by proxy if you have a demoproxy, otherwise uses the default QW recording
64 Recording by the built-in recording in QW.
96 Always records demos, even on normal servers. Useful if you want demos splitted at mapchanges.
Disable Centerprint Turn off most centerprinted messages in Pure. cprint 0 Centerprint on.
128 Centerprint off.
Spectator/observer tracking Selects the style you're tracking with. cam 0 Chasecam.
256 Static Cam.
768 Cam off.
Statusbar HUD support If enabled this option prints the status on two lines which may help if you play with the QW HUD or with cl_showfps enabled. stathud 0 Single line statusbar.
1024 Double line statusbar.
Statusbar Auto ID players Enables "idplayer" as with impulse 141 but automatic. statautoid 2048 Auto ID players.
Statusbar weaponstatus. Shows your current weapon on the statusbar. Not recommended if you are a HPB, since this means the statusbar is updated every time you change weapons. statweapon 4096 Weaponstatus on.
Statusbar partial disable. All these options disables a part of the standard statusbar, which saves screen space and bandwidth. statrune 8192 Disable runestatus.
stattime 16384 Disable time left.
statscore 32768 Disable scores.
Block teamaudio. No Teamaudio is sent to you. (Useful if you don't have the sounds.) teamaudio 65536 Disable Audio.
